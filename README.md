# emergency-knowledge-base-vector-db
# 应急管理知识库系统（基于向量数据库）
> 系统分析与设计 | 向量数据库 | RAG | 应急知识检索

本项目面向**应急管理领域**构建智能知识库系统，基于向量数据库实现应急文本向量化存储、语义检索、相似案例匹配、应急方案推荐等功能。

## 仓库地址
https://github.com/hanghang050928-collab/emergency-knowledge-base-vector-db

## 技术栈
- 向量数据库：Milvus / Chroma / FAISS
- 嵌入模型：Sentence-Transformers
- 后端框架：Python + FastAPI
- 开发语言：Python 3.11+
